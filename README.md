# Prerna Collective

A frontend-only, data-driven site inspired by impact + venture storytelling.  
No backend - all content comes from JSON / MDX files in `public/data`,
rendered via **React (Vite)** + **styled-components**, and
deployable to **GitHub Pages**.

-   **GitHub:** [https://github.com/a2rp/prerna-collective](https://github.com/a2rp/prerna-collective)
-   **Live:** [https://a2rp.github.io/prerna-collective/](https://a2rp.github.io/prerna-collective/)
